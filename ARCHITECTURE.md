# OpenMemory Architecture

The previous mixed architecture has been retired from the active tree.

Current direction:

- JavaScript/Node-first server package in `packages/openmemory-js`.
- Durable cognitive graph architecture.
- Postgres + pgvector as the production storage target.
- Small public API centered on remember, recall, explain, consolidate, and contradiction resolution.

See `docs/architecture-rewrite-plan.md` for the current rewrite plan.
