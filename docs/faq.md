# FAQ

## What is the active product path?

`packages/openmemory-js` is the active product package.

## What runtime is supported right now?

Node.js with TypeScript source and npm scripts.

## What storage target should new work assume?

Postgres with pgvector.

## Are editor extensions, dashboards, and secondary SDKs supported right now?

No. They are deferred until the JS server path and durable core are stable.
