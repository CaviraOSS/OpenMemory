# Core Concepts

OpenMemory is being rewritten toward a durable cognitive graph.

## Current Direction

- JavaScript/Node-first server package.
- Postgres + pgvector production storage target.
- Append-friendly memory records with provenance and auditability.
- Recall modes that separate strict, historical, and associative use cases.
